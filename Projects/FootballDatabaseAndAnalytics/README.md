# Football game database and analytics Project
This project started as a passion project of mine back when I was a kid in 2012. The FIFA game series
was one of the few games I played. With each new game I kept being nostalgic for previously played career modes.

The first version of this project was based in Excel, I wrote down my player statistics attributes et cetera in Excel.

I did it this way until +/- 2023 and took a year break from the series. 

In 2025, I got back into the game and this time I switched from my PlayStation to a PC, this change also opened 
possibilities for me. I wanted to get more hands-on experience with Python outside my work.

It started with converting the data from Excel to Python code, because I kept my Excel in the same format (mostly) over
11 years this wasn't that big of an issue. 

Now with the players in Python I was able to play around with the data alot more than when it was all Excel based.
I created some small Python scripts and then let the project rest.

In 2026, I completed a course in data science using SQL and Python and I saw this project to be the perfect fit to try
and implement my new skills on. 

## Goals
My goals in this project were:
- Gain more experience with Python outside of use cases of my work.
- Get experience with data science tools in Python (Pandas, Matplotlib for example).
- Get experience with (My)SQL.
- Get experience with data visualization

## Reflection
I feel that in this project I'm still missing an in-depth analysis of my data, I have done a couple of small analysis 
inside the Notebooks but nothing to spectacular. This may be because of the lack of "complex" data present or just me not
thinking of something that I can analyze with the data/not having an actual goal for analyzing the data.

But there is still alot to do/learn with this project, I had some gripes with tables and graphs that get outputted with 
the Python packages. 

Because of this I'm currently playing around with Tableau to display the data more interactive and make it easier for a
user to get information from it. (see this [link](https://public.tableau.com/app/profile/martijn.hoek/viz/CareerLegacy_workbook/Dashboard1)
for my current dashboard)
